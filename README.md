# Flask Greeting Project

Мой первый проект на Flask. Простой API для практики Python.

## Что умеет

- `/` - главная страница
- `/greet` - простое приветствие
- `/greet/<name>` - приветствие по имени
- `/api/greetings` - получить все приветствия (GET) или создать новое (POST)
- `/about` - обо мне

## Как запустить

```bash
# Установить Flask
pip install -r requirements.txt

# Запустить
python main.py


```markdown
Открой http://localhost:5001 в браузере
